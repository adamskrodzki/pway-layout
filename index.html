What is PWay?

The primary reason for the PlayWay ICO is to create token value to promote a more community-oriented way to create and publish games.
Our vision is for a worldwide system through which people everywhere will be able to provide their own game ideas and vote for games they want to see developed.

  • We are creating the system to establish a link between players and professional developers in game creation,
  • We will launch our project with an initial coin offering to give our PWay tokens value,
  • The system is based on the Ethereum Blockchain,
  • Rewards for the participants of the project.
